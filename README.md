# 1. 소개
## 1.1. 프로그램 소개
손동작 인식을 통한 단축어 자동 입력 프로그램 (KNN과 OpenCV를 이용한 자주 쓰는 단어 지정)

## 1.2. 사용 기술
- KNN
- openCV
- mediapipe

# 2. Required
## 2.1. python
## 2.2. cv2
```pip install opencv-python ```
## 2.3. opencv-contrib-python
```pip install opencv-contrib-python```
## 2.4. mediapipe
```pip install mediapipe```
